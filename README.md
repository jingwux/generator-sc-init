## How To

### Getting Started

#### 1. 安装yeoman

```bash
$ npm install -g yo
```

#### 2. 安装generator-sc-init

```bash
$ npm install -g generator-sc-init
```

#### 3. 创建你的应用

通过交互模式创建

```bash
$ mkdir {your_project_name}
$ cd {your_project_name}
$ yo sc-init
```

或者使用命令行模式创建

```bash
$ mkdir {your_project_name}
$ cd {your_project_name}
$ yo sc-init -c
```

更多帮助信息可以通过以下命令查看

```bash
$ yo sc-init --help
```
